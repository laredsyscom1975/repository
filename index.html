Operaciones basicas en mysql con php

Profundizaremos aún más con las bases de datos MySQL, aprenderemos sobre las operaciones CRUD en bases de datos.
CRUD es el acrónimo de las palabras en inglés (Create,Read,Update,Delete) en español serían (Crear,Leer,Actualizar,Borrar). Veremos cómo buscar información en las tablas de la base de datos.
Aprenderemos a modificar registros, aprenderemos también a filtrar registros dentro de nuestras consultas.
Borraremos registros dentro de nuestra tabla.
Decimos PHP y MySQL pero en realidad vamos a trabajar con MySQLi que es simplemente una extensión de MySQL, es una version mejorada de MySQL ya que algunas funciones de MySQL quedaron obsoletas. MySQLi es más seguro. No cambiaremos nuestra forma de trabajar, así que no debes preocuparte.
